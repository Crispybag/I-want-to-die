\doxysubsection{Detailed Description}
Static class that contains various settings, such as screen resolution and player controls. In your Main method, you can Call the \mbox{\hyperlink{class_g_x_p_engine_1_1_settings_a1392cbe4ad2d3d891ce531b54321d8a8}{Settings.\+Load()}} method to initialize these settings from a text file (typically settings.\+txt, which should be present in the bin/\+Debug and/or bin/\+Release folder). 

The purpose is that you can easily change certain settings this way, without recompiling the code, and that during development different people can use different settings while working with the same build.
