\doxysubsection{Detailed Description}
Simple \mbox{\hyperlink{class_g_x_p_engine_1_1_mouse_handler}{Mouse\+Handler}} interface between the \mbox{\hyperlink{class_g_x_p_engine_1_1_input}{Input}} class and your own code, that turns the \mbox{\hyperlink{class_g_x_p_engine_1_1_input_a6bfaba5a35563f4e7de3e34d13bf609c}{Input.\+mouseX}} \mbox{\hyperlink{class_g_x_p_engine_1_1_input_a23112cdee5c7b6a5cfa06708a5b453ec}{Input.\+mouseY}} into a more event based approach.

Do not forget to unsubscribe manually from each event before you set the handler to null.
